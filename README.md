📟 AI_TERM (v0.1 Alpha)

AI_TERM — это консольный интерфейс для работы с нейросетями прямо из вашего терминала. Проект объединяет Google Gemini и Mistral AI, предоставляя быстрый доступ к ИИ без необходимости открывать браузер.

✨ Основные возможности

    Dual-Model Support: Мгновенное переключение между Gemini и Mistral.

    Session Memory: Автоматическое сохранение истории диалогов в папку sessions/ [1].

    Cross-Platform: Родная поддержка Arch Linux и готовые сборки для Windows.

    Privacy First: Ваши API-ключи хранятся в локальном .env и не покидают вашу машину.

🚀 Быстрый старт (из раздела Releases)

🪟 Для Windows

    Скачайте AI_TERM.exe.

    Запустите двойным кликом. При первом запуске программа попросит ввести ваши API-ключи.

🐧 Для Linux (Arch/Ubuntu/Debian)

    Скачайте AI_TERM_linux.

    В терминале дайте права на запуск: chmod +x AI_TERM_linux.

    Запустите: ./AI_TERM_linux.

🛠️ Установка из исходного кода (для разработчиков)
На Arch Linux

git clone 
cd AI_TERM
chmod +x install.sh run.sh
./install.sh
./run.sh
