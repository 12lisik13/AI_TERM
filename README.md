# ⚡ ARCH_AI // TERMINAL v0.1

Профессиональный интерфейс для работы с моделями Mistral и Gemini. 
Оптимизировано для **Arch Linux** (2026).

## ✨ Особенности
- **Безопасность**: Ключи API никогда не хранятся в коде (используется `.env`).
- **Сессии**: Автоматическое сохранение истории диалогов в папку `sessions/`.
- **Интерфейс**: Кастомный GUI на Tkinter в стиле Cyberpunk/Gruvbox.
- **Интеллект**: Поддержка Web-поиска и управления контекстом (последние 10 сообщений).

## 🛠️ Установка (Arch Linux)

1. **Клонируйте репозиторий** (или просто скачайте папку с проектом).
2. **Запустите скрипт установки**:
   ```bash
   chmod +x install.sh
   ./install.sh
